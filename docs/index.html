HelloReply

HelloReply is the response returned by SayHello.

FieldTypeLabelDescription
message string

message is the human-readable greeting text.

status HelloReply.Status

debug_id string optional

debug_id is an optional correlation token for tracing individual requests.

metadata HelloReply.ReplyMetadata

follow_up string

A suggested follow-up prompt.

raw_payload bytes

Opaque binary payload for debugging.

HelloReply.ReplyMetadata

ReplyMetadata carries server-side telemetry attached to each response.

FieldTypeLabelDescription
generated_at_epoch_ms int64

Unix timestamp (ms) when the reply was generated.

server string

Hostname of the server that handled the request.

HelloRequest

HelloRequest carries caller identity and contact preferences for a SayHello call.

FieldTypeLabelDescription
name string

name is the caller's display name. Required; an empty value returns INVALID_ARGUMENT.

title string optional

title is an optional honorific (e.g. "Dr", "Ms").

preferred_language HelloRequest.Language

client HelloRequest.ClientInfo

email string

phone_number string

HelloRequest.ClientInfo

ClientInfo identifies the calling application for observability.

FieldTypeLabelDescription
app_name string

e.g. "my-service"

app_version string

e.g. "1.4.2"

HelloReply.Status

Status indicates whether the request was handled successfully.

NameNumberDescription
STATUS_UNSPECIFIED 0

Should not be used.
OK 1

Request processed successfully.
INVALID_ARGUMENT 2

Request was rejected due to invalid input.

HelloRequest.Language

Language controls the greeting language in the reply.

NameNumberDescription
LANGUAGE_UNSPECIFIED 0

Default; server falls back to ENGLISH.
ENGLISH 1

SPANISH 2

PUNJABI 3

HelloService

HelloService provides a simple greeting RPC.

Method NameRequest TypeResponse TypeDescription
SayHello HelloRequest HelloReply

SayHello returns a greeting for the given HelloRequest.
